# Demo Script (5-10 min)

## 1. Intro (30s)

- Present project goal: AI-generated real-time multiplayer quiz battle.

## 2. Teacher Flow (2 min)

- Open `/teacher`
- Paste lesson text
- Generate quiz
- Show generated game code

## 3. Student Flow (3 min)

- Open `/join`
- Enter code and name
- Play live round
- Show timer, streak, XP, effects, lives

## 4. Leaderboard & Persistence (1.5 min)

- Show `/leaderboard/:code`
- Refresh during game and show session recovery

## 5. Technical Proof (2 min)

- Show backend endpoint list
- Show CI status in GitHub Actions
- Show deployment URLs
